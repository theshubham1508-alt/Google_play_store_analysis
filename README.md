# Google_play_store_analysis
Data Analysis Project on Google Play Store Dataset using Python, Pandas and Matplotlib. Includes data cleaning, missing value handling, duplicate removal, exploratory data analysis (EDA) and data visualization.

## Tools Used
- Python
- Pandas
- Matplotlib

## Analysis Performed
- Missing Value Handling
- Duplicate Removal
- Rating Analysis
- Category Analysis
- Free vs Paid Apps Analysis
- Data Visualization

## Dataset
Google Play Store Dataset
